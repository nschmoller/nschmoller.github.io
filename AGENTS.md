# Project Agent Notes

## Run locally

1. Install dependencies:

```bash
npm install
```

2. Start the dev server (BrowserSync):

```bash
gulp dev
```

Run `gulp dev` in a separate terminal and keep it running.

## Build assets

```bash
gulp css
gulp js
```

## Notes

- This is a static site. Main entry is `index.html`.
- Gulp tasks are defined in `gulpfile.js`.
